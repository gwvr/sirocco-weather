# Add pollen forecast from Open-Meteo air quality API

Open-Meteo has a free air quality API (air-quality-api.open-meteo.com) providing hourly pollen data for alder, birch, grass, mugwort, olive, and ragweed (Grains/m³). Europe-only, 4-day forecast window. Source: Copernicus CAMS ENSEMBLE. Show the most relevant UK types (grass, birch, alder) in the hourly table. Should be optional/configurable per location in YAML.
