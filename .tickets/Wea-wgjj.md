# Show temperature as proportional colour bar in hourly table

In the hourly Temp and Feels Like rows, fill the cell background colour only to a height proportional to the temperature relative to the day's min/max (i.e. a mini bar chart effect). The temperature text should be anchored near the bottom of the cell so it always sits on the coloured portion. This makes it visually easy to see the temperature curve across the day.
